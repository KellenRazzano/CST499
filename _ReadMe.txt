Steps to run this project:

1. Download and install xampp from https://www.apachefriends.org/download.html
2. Copy 'CST499_classportal' folder to 'C:\xampp\htdocs\'
3. In phpMyAdmin, create a database named 'registration'
4. Import registration.sql to 'registration' database
5. Open 'http://localhost/CST499_classportal/index.php' on your browser
